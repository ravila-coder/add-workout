Conectar con Asana

La sincronización envía las tareas del Workout a un proyecto de Asana (título, descripción, acreditada, responsable, prioridad, fecha compromiso y estatus de completada) y actualiza las ya enviadas. Necesitas dos datos: un Personal Access Token (Asana → Configuración → Apps → Developer apps → Create token) y el ID del proyecto (el número largo en la URL del proyecto, p. ej. app.asana.com/0/1205551234567890/list).

Sin conexión configurada

El token se guarda en el almacenamiento local de este navegador; cualquiera con acceso a este dispositivo podría usarlo. Para revocarlo, elimínalo desde Asana. Si el sitio corre dentro de un entorno con red restringida (como la vista previa de este chat), la sincronización debe hacerse desde la versión desplegada en Vercel.
